# Task 1 - Data Preprocessing

## Description
Data preprocessing was performed on the Churn Prediction dataset to prepare it for machine learning.

## Steps Performed
- Checked and handled missing values
- Encoded categorical variables (Label Encoding)
- Standardized numerical features (StandardScaler)
- Split data into 80% train and 20% test sets

## Dataset
- churn-bigml-80.csv (Training data)
- churn-bigml-20.csv (Testing data)

## Tools Used
- Python
- Pandas
- Scikit-learn
