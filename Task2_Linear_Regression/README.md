# Task 2 - Linear Regression

## Description
Built a Linear Regression model to predict house prices using the Boston Housing dataset.

## Steps Performed
- Loaded and parsed space-delimited dataset
- Added proper column names
- Checked missing values
- Separated features (X) and target variable (MEDV - house price)
- Split data into 80% train and 20% test sets
- Standardized numerical features (StandardScaler)
- Trained Linear Regression model
- Evaluated model using R2 Score and MSE
- Visualized Actual vs Predicted prices

## Results
- R2 Score: 0.67
- MSE: 24.29

## Dataset
- house Prediction Data Set.csv (Boston Housing Dataset)

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
